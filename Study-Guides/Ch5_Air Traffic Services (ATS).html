Top Exam Traps
🪤
Class E CANNOT be used for a Control Zone
Class E is IFR-only separation — specifically prohibited from CTR designation. Control Zones must use Class A, B, C or D.
🪤
Class C: VFR NOT separated from other VFR
Class C separates IFR↔IFR and IFR↔VFR only. VFR↔VFR gets traffic information — not separation. VFR↔VFR separation only in Class B.
🪤
7500=Hijack · 7600=Radio · 7700=Emergency
Memory: 75=Taken Over · 76=Can't Talk · 77=Need Help. Examiner swaps 7500 and 7700 in options — know these cold.
🪤
Holding aircraft = ARRIVING (not en-route)
Any aircraft holding near an aerodrome is classified as an ARRIVING aircraft for ATC priority — frequently asked in DGCA orals.
🪤
TSA = Nobody enters · TRA = ATC may allow
TSA (Segregated) = completely excluded. TRA (Reserved) = transit with ATC clearance. Segregated is stricter. Students often swap these.
🪤
VOR=3 letters · Significant Point=5 letters
VOR = exactly 3 (AAT, DEL). Significant point/waypoint = exactly 5 (GOLEM, IDOLA). ILS = 4 starting with I. Don't mix VOR(3) with Sig Point(5).

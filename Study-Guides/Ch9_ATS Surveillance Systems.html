⚠️ Top Exam Traps
🪤
PSR = Position ONLY (no altitude, no identity)
PSR works purely by radar reflection — it cannot read transponders. No squawk code, no altitude. SSR provides all three: Position + Identity + Altitude.
🪤
ADS-B = Everyone · ADS-C = Only contracted ATSU
ADS-B Broadcast = all receivers get the data. ADS-C Contract = only the specific ATSU/AOC that has a contract with the aircraft receives the data.
🪤
Turn method = 30° minimum (not 20° or 45°)
For radar identification by turn method, ATC instructs a heading change of at LEAST 30°. Only one radar target should make the same turn.
🪤
No code = Squawk 2000 (not 7000 or 1200)
India: Aircraft without assigned discrete SSR code must squawk 2000 before entering Indian FIRs. 7700=Emergency, 7600=Radio fail, 7500=Hijack.
🪤
Stop Squawk Charlie = Stop MODE C only (not entire transponder)
"Stop Squawk Charlie" means stop only the altitude (Mode C) transmission. The identity (Mode A) continues squawking. NOT a full transponder shutdown.
🪤
220 kt only within 15 NM (not everywhere below 10,000ft)
The 220 kt limit is ONLY for arriving aircraft within 15 NM of VOR/DME. The general limit below 10,000ft is 250 kt.
